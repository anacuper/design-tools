Argon + A-Frame Examples

The A-Frame javascript library lets you create WebVR applications using custom DOM elements, allowing 3D content to be manipulated similarly to 2D HTML elements.

We have adapted A-Frame to work with Argon, creating a new ar-scene element and a number of entities and components to support AR. The source and programmer documentation is on the github page (click on the ribbon in the upper right corner). The developer documentation is here

In order to see these examples work properly, you must view them in the Argon browser. If you are viewing this page in an ordinary browser, then copy the url in the address window above and paste it in the Argon browser (downloaded from the App store).
